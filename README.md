# Nanashi 🚀🔒🐝

**IA souveraine on-device – Privacy absolue, sans nom, sans trace**

Nanashi (無名 – "sans nom" en japonais) est une intelligence artificielle locale, 100 % on-device, optimisée exclusivement pour Apple Silicon (M-series : Mac Mini Pro, MacBook, iPhone, iPad).

Aucune donnée ne quitte jamais votre appareil. Zéro cloud, zéro fuite.

### Fonctionnalités principales
- **Privacy absolue** : Split inference avec injection de bruit additif – vos données restent sur l'appareil, jamais exposées.
- **Nanashi Consensus** : Proof of Intelligence anonyme (préparation future subnet Bittensor privacy).
- **Performance native** : Inférence ultra-rapide via Ollama + MLX (framework Apple) – Llama 3.1, Mistral et modèles ouverts à pleine vitesse Neural Engine.
- **UI premium** : Interface neon/glassmorphism responsive (dashboard mining simulé, chat sécurisé, audit sécurité, settings).
- **Mode hybride** : Local souverain + contribution anonyme décentralisée (Bittensor / io.net en roadmap).

### Pourquoi Nanashi ?
- Privacy > tout (contrairement aux concurrents semi-ouverts ou cloud-dépendants).
- Exclusif Apple Silicon (marché massif inexploité en IA décentralisée).
- Open-source MIT – libre à utiliser, modifier, déployer.

### Démo instantanée (Mac M-series)

# 1. Installer Ollama

# 2. Lancer Ollama en arrière-plan
